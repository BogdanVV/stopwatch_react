Обычный секундомер на реакте.
Кнопка startзаменяется на стstopоп при нажатии, и наоборот.
Кнопка 'wait' работает на дабл-клик (валидный интервал между кликами >300ms). При нажатии на start отсчет продолжается.
Reset обнуляет таймер, но не останавливает.
Седално с целью практики и дальнейшего внедрения RxJs.
